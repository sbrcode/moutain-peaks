# moutain-peaks

## installation
(python 3.6.6, pip and sqlite3 were already installed)

<ul>
  <li>python -m pip install Django</li>
  <li>django-admin startproject apirest</li>
  <li>python manage.py runserver</li>
  <li>python manage.py startapp polls</li>
</ul>

## execution

<ul>
  <li>python manage.py runserver</li>
</ul>
